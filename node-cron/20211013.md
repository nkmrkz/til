# Node.jsの定期実行の方法
## node-cronを使う
インストール
```
yarn add node-cron
```

index.js
```
import cron from "node-cron"

const main = ()=>{
  console.log(new Date)
  console.log("hello")
}

cron.schedule("55 * * * *",()=>{
  main()
})
```

- scheduleメソッドの第一引数はcrontabで定義するスケジュールと同じ書式

実行
```
% node src/main.js   
2021-10-13T14:55:00.129Z
hello
```
